--------- GoLang Basics to Advance ----------



Basic commands :

go run file_name.go 		: to run your go file
go help 					: to list all common commands and additional commands
go help command_name 		: to get the documentation or brief overview about the command
go mod init text_name 		: 
go mod tidy					:
go env 						: to find all environment variables configuration
go build                    : to build a executable file for the current operating system
GOOS="OS_name" go build     : to build a executable file for the any operating system
